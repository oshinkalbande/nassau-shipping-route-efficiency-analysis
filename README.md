# Nassau Candy Distributor - Shipping Route Efficiency Analysis

## Project Overview

This project analyzes factory-to-customer shipping routes for Nassau Candy Distributor.

### Objectives
- Identify efficient shipping routes
- Detect bottleneck states
- Compare ship modes
- Build an interactive Streamlit dashboard

## Technologies Used
- Python
- Pandas
- Plotly
- Streamlit
- Scikit-Learn
- Matplotlib
- Seaborn

## Features
- KPI Dashboard
- Route Efficiency Leaderboard
- Geographic Shipping Analysis
- Ship Mode Comparison
- Interactive Filters

## How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the dashboard:

```bash
streamlit run app.py
```
